# HACK-TECH-TALK

Hackathon workspace for **Community Garden** — a web and mobile marketplace connecting local producers (home gardeners and small farmers) with consumers and brokers.

Key features:
- **AI natural-language search** — consumers find produce by typing plain prompts ("Find me some zucchini"); LLM extracts intent and returns nearby available listings.
- **Future Orders** — consumers post a demand signal with quantity and time window ("10 oranges in 2 days"); notified the moment a matching producer listing appears within their proximity.
